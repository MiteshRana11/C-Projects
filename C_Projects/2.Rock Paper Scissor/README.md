# Rock Paper Scissors Game (C)

This project is a simple console-based Rock Paper Scissors game written in C where the user plays against the computer.

## Features

* User vs Computer gameplay
* Random computer choice
* Game result display
* Simple command-line interface

## Concepts Used

* Random number generation
* Conditional statements
* Functions
* Loops

## How to Run

1. Compile the program

gcc RPS.c -o game

2. Run the program

./game

## Learning Outcome

This project demonstrates the use of random numbers, decision making, and basic game logic in C programming.
