# Library Management System (C)

This project is a simple console-based Library Management System written in C.
It helps manage book records and demonstrates how file handling can be used to store and retrieve data.

## Features

* Add new book records
* View all books
* Search book by ID
* Delete book records
* Store data using file handling

## Concepts Used

* Structures
* File Handling
* Functions
* Menu Driven Programming
* Basic Data Management

## How to Run

1. Compile the program

gcc LMS.c -o library

2. Run the program

./library

## Learning Outcome

This project helped me understand how basic data management systems work using C programming concepts.
