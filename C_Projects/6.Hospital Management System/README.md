# Hospital Management System (C)

This project is a console-based Hospital Management System written in C.

## Features

* Add patient record
* View patient details
* Search patient by ID
* Delete patient record
* Data stored using file handling

## Concepts Used

* Structures
* File Handling
* Functions
* Menu Driven Programming

## How to Run

gcc HMS.c -o hospital
./hospital

## Learning Outcome

This project helped me understand how real-world systems store and manage records using C programming.
